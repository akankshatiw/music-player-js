
# music player 

A modern Spotify-inspired music player built using HTML, CSS, and JavaScript.

This project uses the iTunes API to dynamically fetch songs, display artists, and play music previews with a clean glassmorphism UI design.

---

## Features

-  Search songs dynamically
-  Popular artist section
-  Add songs to favorites
-  Play/Pause music previews
-  Fetch API integration
-  Async/Await functionality
-  Glassmorphism UI
-  Responsive layout
-  Spotify-inspired footer player

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API
- Async/Await
- iTunes Search API
